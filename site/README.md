# Сайт заказов ALABAR

Каталог и наличие подгружаются с API склада.

## Подключение к API

- По умолчанию запросы идут на тот же хост (`/api/public/items/`). Если сайт раздаётся с того же сервера, что и Django (например, через nginx или `python -m http.server` в папке с `index.html` на том же порту), ничего настраивать не нужно.
- Если API на поддомене (например **api.kchrmarket.ru**), в `index.html` в `<head>` раскомментируйте:  
  `<script>window.ALABAR_API_BASE='https://api.kchrmarket.ru';</script>`
- Для локальной разработки: в консоли `window.ALABAR_API_BASE = 'http://127.0.0.1:8000';` и обновите страницу.

## Эндпоинт

- `GET /api/public/items/` — список товаров с остатками (без авторизации).
- Опционально: `?workshop_id=uuid` — только товары указанного цеха.

## Наличие

- В карточке товара: «В наличии» / «Нет в наличии».
- В модалке товара: по каждому размеру показывается остаток («56 — 3 шт.» или «56 — нет»). В корзину можно добавить только размеры с остатком > 0.
